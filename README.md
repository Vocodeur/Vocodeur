Vocodeur
========
Pour récupérer des valeurs de fichiers sons, le format .wav est pas mal.
Il est consititué juste d'une en-tete, et après on a les différentes valeurs prises au cours du temps (éventuellement 2 canaux enlacés). C'est donc relativement simple à récupérer.
cf http://www-mmsp.ece.mcgill.ca/Documents/AudioFormats/WAVE/WAVE.html pour les explications concernant cet encodage.
